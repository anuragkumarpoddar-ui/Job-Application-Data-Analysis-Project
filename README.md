# Job-Application-Data-Analysis-Project
🚀 Overview

This project focuses on analyzing job application data using Python and Pandas to extract meaningful insights.
It simulates a real-world job tracking system where applications are monitored, analyzed, and optimized using data-driven techniques.

The goal is to understand:

Application success trends
Platform effectiveness
Interview conversion rate
Salary insights
🛠️ Tech Stack
Python
Pandas
Matplotlib (for visualization)
Jupyter Notebook / VS Code
Git & GitHub

📁 Dataset Description


The dataset contains structured job application records with the following fields:

Column Name	Description
Company	Company name
Role	Job role applied
Location	Job location
Platform	Source of application (LinkedIn, Naukri, Referral)
Date_Applied	Date of application
Status	Applied / Interview / Rejected
Salary_Offered	Offered salary (if any)
⚙️ Key Operations Performed
🔹 Data Preparation
Created dataset using Python dictionary
Converted date column into datetime format
🔹 Exploratory Data Analysis (EDA)
Data overview using .info() and .describe()
Status distribution analysis
Platform usage analysis
🔹 Data Analysis
Total applications count
Interview conversion rate
Average salary (excluding zero values)
🔹 Grouping & Aggregation
Platform vs Status analysis
Applications by location
Average salary by role
🔹 Filtering
Extracted interview calls
Identified high-paying jobs (>7 LPA)
🔹 Sorting
Ranked jobs based on salary
📈 Key Insights
📌 Total Applications: 7
📌 Interview Rate: ~57%
📌 Highest Salary Offer: ₹12 LPA
📌 Top Platforms: LinkedIn & Referral
📌 High Opportunity Locations: Bangalore & Hyderabad
